# TerraEngine-project
Github repo for TerraEngine, a simple and bare-bones game engine based on a rooms architecture

#About
-----------------------------------
Made in c++ and natively meant for the linux platform
Windows support is not guaranteed by RauXi, and is an edge case

The engine architecture consists of "rooms" that act as separate areas for the game
The main "engine" referred is the components of the project that handle ingame logic
The map builder, which may be confused for the engine, is referred to as the map builder, room parses, or map editor

Engine is not meant to support advanced materials and uses .obj and .glb format for files

#Versions
-----------------------------------
v.V.V.V-a to be released when the map editor is properly ready. Any other version is to be named v.0.0.0-a-[system] eg. v.0.0.0-a-text or any other implemented component
